DOCUMENTATION

In this section you can learn how to create an agent for JSWars, in your profile page you can add new agent codes writes on Javascript. Just type a name for your agent and the code with the following structure, it must have a function init with the code that you want to execute before the battle start, in this point you have all the battle information about map units, and all of his attributes. You have a longer timeout for compute that info and use it to prepare whatever you want for the battle.

After that, the battle begins and in every thick of the game the game will execute your function tick(), it will have a shorter time out to execute this. This function must return a list of actions for your units, you can move, shoot or both actions with any of your units. If you exceed the timeout for return an action your units will not move or shoot.

Utils.colMap

In this section you have access to all Map information, width and height of the map, the colMap consist in a grid with the collision information, and there is a function with a pathfinding using A* accesible for the agents, if you wanna implements yours you're able to do so.

In the colMap array for check a position collision you must get colMap[x][y], the value 0 means free collision and 1 means collision. Be sure to apply a floor function with the position to get the info without errors.
